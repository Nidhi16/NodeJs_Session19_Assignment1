I am using linux machine. So, here the steps which I follow

1) Download the linux based node webkit sdk
2) Make package.json and index.html file
3) Comress both files and name it app.nw
4) Now run the application -

                3) Place the app.nw in the nw folder
                4) Type ./nw app.nw
